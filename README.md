# 视频分类脚本

将视频文件按"开头"进行分组，确保每个目标文件夹内的视频"开头"全部不相同。

## 功能特性

- 自动识别视频文件名中的"开头"信息
- 支持有剧名和无剧名两种格式
- 预览模式：先查看分配结果，确认无误后再执行
- 跨平台支持：Windows / Mac / Linux
- 无需安装任何第三方库

## 快速开始

### Mac 用户（推荐）

1. 将以下文件放到视频所在文件夹：
   - `视频分类脚本.py`
   - `预览.command`
   - `执行.command`

2. 双击 `预览.command` 查看分配结果

3. 确认无误后，双击 `执行.command` 执行移动

### Windows 用户

```bash
# 预览
python 视频分类脚本.py "C:\视频目录"

# 执行
python 视频分类脚本.py "C:\视频目录" -e
```
